To run this:

1) You need special drivers: https://devblogs.microsoft.com/directx/cooperative-vector/#get-running
2) You need to turn on developer mode: https://learn.microsoft.com/en-us/windows/apps/get-started/enable-your-device-for-development
3) Run the viewer with this command line before loading the technique: -AgilitySDKPreview -nopixcapture -norenderdoc

For more information, please see UserDocumentation\CooperativeVectors.docx / .pdf in the Gigi root folder.
